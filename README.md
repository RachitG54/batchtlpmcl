# liblhtlp - C Library implementing Linearly Homomorphic Time Lock Puzzles

This is an implementation of [Homomorphic Time-Lock Puzzles and
Applications](https://eprint.iacr.org/2019/635.pdf) - Section 4.1

## Dependencies

- mcl
- `gcc v9.2.1`
- `gmp v6.2.0-1`
